Assigning VHs - PS


Old VHs - PS

  • - Copy the case number.
  • - Search in the excel sheet.
  • - look for previous processors.
  • - Check for their availability.
  • - If available: re-assign
  • - If chat support: re-assign.
  • - If not available: Look for available processor.

New VHs - PS

  • - Find new available processor.
  • - Check Excel sheet for fair distribution.
  • - Check RCC availability.
  • - Assign in the ticket.
  • - Update Excel sheet.
  • - Inform team in MS Teams.
