Assigning VHs

Assigning VHs - PS

Old VHs:

  • 🎀 Copy case number.
  • 🎀 Check for previous processors in excel sheet.
  • 🎀 Check previous processors availability in RCC.
  • 🎀 If available --> Re-assign.
  • 🎀 If chat support --> Re-assign.
  • 🎀 If not available --> Look for other processor.

New VHs:

  • 🎀 Find new available processor.
  • 🎀 Check Excel sheet for fair distribution.
  • 🎀 Check RCC availability.
  • 🎀 Assign in the ticket.
  • 🎀 Update Excel sheet.
  • 🎀 Inform team in MS Teams.

We do not assign DS VHs 🎀
